History

How did we get from JavaScript to TypeScript?

JavaScript

  • Developed by Netscape / Brendan Eich
  • 1995
  • September - LiveScript
  • December - JavaScript

EcmaScript

  • Netscape asked Ecma International to create specification
  • Started in 1996, first release in 1997
  • First version of ECMAScript was based on JavaScript
  • All implementations are based on ECMAScript
  • Only defines Core of the language
  • DOM is maintained by W3C

TypeScript

  • Published in 2012
  • Anders Hejlberg
    • Lead architect for C#
    • Delphi en Turbo Pascal
    • ContributedTypeScript
  • Made for large-scale apps
  • Superset of JavaScript
    • Existing JavaScript apps are TypeScript apps

JavaScript

Scope and "this"

Scope

  • Java has block-level scoping
  • JavaScript has function-level scoping
  • Variables assigned in a function are not visible outside
  • A variable defined anywhere in a function is visible everywhere within the function
  • Also has impact on "this"

Prototypes

		
function Person(first, last, age, eyecolor) {
    this.firstName = first;
    this.lastName = last;
    this.age = age;
    this.eyeColor = eyecolor;
}
Person.prototype.name = function() {
    return this.firstName + " " + this.lastName;
};
Person.prototype.getAge = function() {
    return this.age;

TypeScript is a superset of ES6

Features

  • Type annotations
  • Public/Private
  • Compile-time type checking
  • Type inference
  • Interfaces
  • Enums
  • Mixin
  • Generics
  • Optional properties
  • Tuple types

Bottom Line

  • If you are used to Java and C#, you should feel at home in TypeScript
  • If you hate looking at Type information, you should not use TypeScript
  • If you believe in compile-time checking, use TypeScript
  • If you have problems with Type-related bugs in JavaScript, use TypeScript
  • If you are not goint to make use of Type annotations, just use ES6

Intersection type

		
function extend<T, U>(first: T, second: U): T & U {
    let result = <T & U>{};
    for (let id in first) {
        (<any>result)[id] = (<any>first)[id];
    }
    for (let id in second) {
        if (!result.hasOwnProperty(id)) {
            (<any>result)[id] = (<any>second)[id];
        }
    }
    return result;
}

class Person {
    constructor(public name: string) { }
}
interface Loggable {
    log(): void;
}
class ConsoleLogger implements Loggable {
    log() {
        // ...
    }
}
var jim = extend(new Person("Jim"), new ConsoleLogger());
var n = jim.name;
jim.log();

Union type

		
function doSomething(something: string | number) {

   if (typeof something === "number")
      return Array(something + 1).join(" ");

   if (typeof something === "string")
      return something;
}

   return new Promise( resolve => {
